## 总结出几条线路

### 📚 元工作（领域坐标系）

```
Gao et al. Survey (2025-07, arXiv:2507.21046) ⭐ ── What/When/How/Where 框架
Fang et al. Survey (2025-08, arXiv:2508.07407) ── 模型/环境/共演化三分法
```

### 🔵 主线 A：权重进化的 RL 路线

```
GRPO (2024-02) ⭐⭐ ━━━━ 算法基石（DeepSeek-Math）
     │
     ├─→ Dr.GRPO (2025-03) ── 修正长度/难度偏差（NUS）
     ├─→ DAPO    (2025-03) ⭐ ── 工程精修四件套（字节-清华 SIA）
     ├─→ GSPO    (2025-07) ── 序列级，MoE 适配（阿里 Qwen）
     └─→ 信用分配分支：GiGPO/SPA-RL/ARPO/SALT/HiPER/HCAPO/ADCA-GRPO
                     （2025-05 至 2026-03，渐进改进，尚未收敛）

并行的 Agent RL 实战：
WebRL (2024-11) ── self-evolving curriculum（THUDM）
RAGEN/StarPO (2025-04) ⭐ ── 多轮 LoRA RL（Northwestern）
```

### 🟢 主线 B：不动权重的反思演化路线

```
DSPy (2023-10) ━━━━ 声明式 LM pipeline 起点（Stanford）
     │
     ├─→ TextGrad (2024-06, Nature 2025) ⭐ ── 文本梯度，登 Nature
     │
     └─→ GEPA (2025-07, ICLR 2026 Oral) ⭐⭐ ── 反思 + Pareto 遗传
              │                              400-1200 rollouts 超 GRPO 24k
              ▼
         ACE (2025-10, ICLR 2026) ⭐ ── playbook 演化，AppWorld 击败 GPT-4.1
              （ACE 直接继承自 Dynamic Cheatsheet，与 ReasoningBank 并行）
```

### 🟣 主线 AB 混合：Memory + RL/Context 闭环（2025 下半年前沿）

```
SEAL (2025-06, NeurIPS 2025) ⭐ ── self-edit 文本 → LoRA 权重（MIT）
     │
ReasoningBank (2025-09, ICLR 2026) ⭐ ── 策略级记忆 + MaTTS（Google + UIUC）【不动权重】
     │       与 ACE 并行竞争，非承继
     │
EvolveR (2025-10, ICLR 2026) ⭐ ── offline 蒸馏 + online GRPO 闭环
     │
Evo-Memory (2025-11) ── 记忆条目持续演化【不动权重】
     │
AgentEvolver (2025-11, Alibaba Tongyi) ⭐ ── Self-Q/N/A + ADCA-GRPO，产业级开源
```

### 🟡 主线 C：科研场景端到端系统

```
NovelSeek (2025-05) ⭐ ── 12 类科研任务，self-evolving idea generation
     │  （上海 AI Lab）
     ▼
InternAgent / NovelSeek 1.5 (2025 下半年迭代) ── GAIA/HLE/GPQA/FrontierScience SOTA
```

### 🟠 主线 D：产业部署与人机协作（非科研）

```
ARIA (EMNLP 2025 Industry) ⭐ ── TikTok Pay 1.5 亿用户，人机协作 RLHF
AITL (2025-10) ── Agent-in-the-Loop 框架
```

### 🟤 主线 E：评估基准演进

```
通用基准：AgentBench (2023) → AgentBoard (2024) → SWE-Bench → xBench (2025)
科研基准：
  ScienceAgentBench (ICLR 2025) ── 数据驱动科研子任务，102 任务 4 学科
  BixBench (2025-03) ⭐ ── 残酷数字：前沿模型 open-answer 仅 21%
  HypoBench (2025-04) ── 假设生成首个原则化基准
  MLR-Bench (2025-05) ⭐ ── 揭露 fabricated experiments
  BioDSA-1K (2025-05)
  REPRO-Bench (2025-07) ── 直击可复现性
  SciAgentGym (2026) ── 259 任务 / 1780 工具 / L1-L3 长时程
  scBench / AIRS-Bench / BioAgent Bench / MoSciBench (2026)
```
