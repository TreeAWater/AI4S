# Self-Evolving Agents Paper Library

This folder stores PDFs from the requested Self-Evolving Agents reading list.

## Organization

- `01_foundations`: foundational papers needed to read the rest of the field.
- `02_paradigm_openers`: papers that introduce major self-evolution paradigms.
- `03_current_sota`: recent flagship methods.
- `04_industry_hitl`: industrial and human-in-the-loop deployments.
- `05_scientific_grounding`: scientific-agent and research-evaluation papers.
- `06_rl_variants`: GRPO/RL descendants and related training variants.
- `07_gradient_free_memory_context`: reflection, context, memory, and test-time evolution.
- `08_workflow_skill_benchmarks`: workflow, skill, and benchmark papers.

`manifest.tsv` records each title, source URL, local filename, and download status. Re-run `python3 doc/download_papers.py` from the repository root to fill missing files or regenerate the manifest.
